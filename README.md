# Tiktok
## 抖音项目服务端简单示例
使用gorm + gin + mysql + redis 编写的极简版抖音，实现了基础接口，社交接口和互动接口的功能。


### 项目答辩文档
https://qjxdqi8snh.feishu.cn/docx/KjOjdmeLho3ZQUxgrhVchB86n8c?from=from_copylink

工程无其他依赖，直接编译运行即可，服务器与数据库配置信息自行修改

```shell
go build && ./tiktok
```
